# Hera Yacht Charter - Versão CNPJ 65.066.161/0001-23

Esta pasta contém uma variação do site da **Hera Yacht Charter** focada em eventos e experiências corporativas, associada ao CNPJ **65.066.161/0001-23**.

## Visão Geral

O conteúdo apresenta serviços de charter de iates voltados para meetings, offsites, ativações de marca e experiências de relacionamento com clientes estratégicos.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Mesmo layout náutico e responsivo da versão base.
-   **Foco:** Segmento corporativo e B2B.
